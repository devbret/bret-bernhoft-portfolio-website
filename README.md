# Bret Bernhoft's Portfolio Website

![Screenshot from the top of the website.](https://hosting.photobucket.com/bbcfb0d4-be20-44a0-94dc-65bff8947cf2/96e23749-2461-4070-ba79-b06087b139f1.png)

A portfolio website for Bret Bernhoft, a Full Stack Software Engineer who specializes in low-code development with Quickbase and cloud automation.

## Overview

The website introduces Bret's background spanning WordPress through enterprise systems, highlights his leadership since 2021 building secure, scalable Quickbase apps and API integrations and emphasizes a user-centric, collaborative approach with real-time reporting.

The site showcases a wide range of featured projects, from a cyberpunk React website and D3 OSINT visualizations to audio analysis tools, transit data mapping, homelab/Kubernetes documentation, Raspberry Pi sensor work and local-LLM experiments. It also includes LinkedIn endorsements attesting to his technical depth and teamwork and provides contact details for opportunities and technical mentoring.
